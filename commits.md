### Commits
- Gostaria de fazer um commit para o projeto? Para um maior entendimento entre os participantes, use o seguinte modelo:
```markdown
[emoji] [arquivo alterado] - [fase do projeto] [numero do commit]
```
- exemplo:
```mardown
 ⬆️ README.md - Fase de Pesquisa #1
 ```

### Tabela de Emojis:
| Emoji|Significado|
|---------|----------|
| 	✅		 |	 Add File
| 	❌		 |   Delete File
| 	⬆️ 		|   Update File
| 	🐛		|	  Bug
| 	📚		| 	Grammar Correction
